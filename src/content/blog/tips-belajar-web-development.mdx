## Mulai dari Mana?

Belajar web development bisa terasa overwhelming karena banyaknya teknologi yang harus dipelajari. Tapi jangan khawatir — kuncinya adalah **mulai dari dasar** dan **konsisten**.

## Roadmap yang Saya Rekomendasikan

### 1. HTML & CSS Dulu

Ini adalah fondasi dari web development. Pastikan kamu paham:

- Semantic HTML (header, main, section, article)
- CSS Flexbox dan Grid
- Responsive design
- CSS custom properties

### 2. JavaScript

Setelah HTML & CSS, pelajari JavaScript:

- Variables, functions, loops
- DOM manipulation
- Async/await dan Promises
- ES6+ features (arrow functions, destructuring, spread operator)

### 3. Framework

Pilih salah satu dan kuasai:

- **React** — paling populer, ekosistem besar
- **Vue** — mudah dipelajari, dokumentasi bagus
- **Svelte** — ringan, DX terbaik

### 4. Backend

Untuk backend, ada beberapa pilihan:

- **Laravel** (PHP) — framework yang sangat lengkap
- **Node.js** (Express/Fastify) — JavaScript di server
- **Next.js** — full-stack React framework

## Tips Praktis

1. **Buat project nyata** — jangan hanya tutorial
2. **Baca dokumentasi resmi** — lebih baik dari tutorial random
3. **Konsisten** — 1 jam sehari lebih baik dari 10 jam sekali seminggu
4. **Join komunitas** — belajar dari orang lain
5. **Jangan takut error** — error adalah guru terbaik

## Resources yang Berguna

- [MDN Web Docs](https://developer.mozilla.org) — referensi HTML, CSS, JS terlengkap
- [freeCodeCamp](https://freecodecamp.org) — kursus gratis dan terstruktur
- [Next.js Docs](https://nextjs.org/docs) — dokumentasi Next.js resmi
